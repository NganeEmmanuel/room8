# Project structure

```
src/
│
├── app/               # Recoil/Context or store config if needed
├── assets/            # Static files/images
├── components/        # Reusable UI components
├── layouts/           # Layout wrappers (e.g., AuthLayout, MainLayout)
├── pages/             # Route-level pages
├── services/          # Axios API config, AuthService.js, etc.
├── hooks/             # Custom hooks
├── utils/             # Helper functions (e.g., token utils)
└── router/            # React Router config

```
